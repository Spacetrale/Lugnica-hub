## What is Lugnica hub?

It's a **scripts hub** for html/js/css games, like Cookie Clicker and more.
When you have the **scripts hub** you can launch the **script** for your game, so you will be able to give you some items or money

## How install it?

It's simple, Go to you're **console** and put this:
```js
fetch("https://raw.githubusercontent.com/Spacetrale/Lugnica-hub/refs/heads/main/lugnica%20hub.js").then((r) => {return r.text()}).then((r) => {eval(r)})
```
And then you have the hub! So after you need to launch the script for your game.

## How launch a script?

When you execute the Lugnica hub you will see some green button, when you click on one of these you have another gui that spawn.
You will have many options for modifying the game. Like autocliker, give money, give items ect...
